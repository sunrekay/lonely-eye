## Запуск
Склонируйте репозиторий:
```
git clone https://github.com/sunrekay/lonely-eye
```
Перейдите в него:
```
cd lonely-eye
```
Смените имя: ".env-example" -> ".env"

Соберите контейнеры:
```
docker compose build
```
Запустите:
```
docker compose up
```
